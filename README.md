Mirror is a compiling, testing and deployment for Hyperledger Besu, aiming to make life as an Enterprise Ethereum developer easier.

## Quick Start

```shell
#Install
$ npm install -g mirror-besu


# Compile 
mirror compile

# Deploy a private contract
mirror deploy --private

```
